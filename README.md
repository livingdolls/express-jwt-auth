## Run Locally

Clone the project

```bash
  git clone https://github.com/livingdolls/express-jwt-auth.git
```

Go to the project directory

```bash
  cd express-jwt-auth
```

Install dependencies

```bash
 npm  install
```

Start the server

```bash
  npm start
```
